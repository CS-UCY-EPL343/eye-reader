▸ saveUserSettings()

This function is responsible for saving the currently logged in user's settings in the device's local storage. It creates a temporary array that holds the user's settings. Then it retrieves the settings of all the users from the local storage, filters out the current logged in user's settings and saves the new settings.

▸ Login(username, password, callback)

This function is responsible for searching all the users' profiles to find the matching username in the users' list. If the username is not found then a failure callback is returned. Else if the username is found, then it checks if the password matches. If it does, it returns a success callback. If not, then it returns a failure callback. Also, it updates the logged in user's profile value: firstTime, in order to keep track if this is the first time a user is logging in.

Login(username: string, password: string, callback: string)
Parameters
username (string) The user's username
password (string) The user's password
callback (string) The response back to the caller
