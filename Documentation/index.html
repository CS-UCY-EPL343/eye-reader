▸ Login(username, password, callback)

Responsible for searching all the users' profiles to find the matching username in the users' list. If the username is not found then a failure callback is returned. Else if the username is found, then it checks if the password matches. If it does, it returns a success callback. If not, then it returns a failure callback.
